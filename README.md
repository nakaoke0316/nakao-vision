# nakao-vision

ナカオ工業 ビジョン共有用 HTML（全16スライド体制）

## 構成

- `index.html` — ビジョンスライドデック本体（自己完結型・JS/CSS埋め込み済み）
- スライド構成：VISION（哲学→骨格→具体）→ STRATEGY（3本柱・並走と段階拡大・DX論・情報ループ・BOM）→ MISSION（部門→製造→中尾宣言→DX組織変革→SaaS構成）→ FUTURE

## 開発

ローカルで開く：

```
start index.html
```

特別なビルド不要。1ファイル完結。

## デプロイ

- Vercel: main ブランチへの push で自動デプロイ
- 公開URL: （Vercel接続後に追記）

## 注意

- モリタエコノス・タイ・セブンイレブン等の固有名詞を含む（公開URL運用）
- 機密度の高い情報は含まない方針
